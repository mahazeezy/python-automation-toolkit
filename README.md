# 🤖 Python Automation Toolkit

> A full-stack web application with 6 real-world automation tools — built with Python & Flask. Live on the web, zero dependencies to install.

## 🚀 Live Demo
**[▶ Try it live](https://mahazeezy.pythonanywhere.com/automation)**

---

## 🛠️ Tools

| Tool | Description |
|------|-------------|
| 📁 **File Organizer** | Paste a list of file names — instantly sorted into folders by type |
| 🌐 **Web Scraper** | Extract links, headings, images, or text from any URL |
| 📝 **Text Processor** | Word stats, extract emails/phones, clean text, word frequency |
| 🔗 **URL Health Checker** | Check if a list of URLs are live or returning errors |
| 📊 **CSV Analyzer** | Paste CSV data and get instant column stats and summaries |
| 🔐 **Password Generator** | Generate strong, medium, or PIN passwords in bulk |

## ⚡ Run Locally
```bash
git clone https://github.com/mahazeezy/python-automation-toolkit
cd python-automation-toolkit
pip install flask
python app.py
```

Then open `http://localhost:5000/automation`
